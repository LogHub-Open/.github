## Reportando uma vulnerabilidade

**Não abra issues públicas para vulnerabilidades de segurança.**

Para reportar uma vulnerabilidade, envie os detalhes diretamente aos mantenedores através das **GitHub Security Advisories** do repositório afetado (aba _Security > Report a vulnerability_).

Inclua:

- Descrição da vulnerabilidade
- Passos para reproduzir
- Impacto potencial
- Sugestão de correção (opcional)

Faremos o possível para responder em até **72 horas** e divulgar um patch de forma responsável.
