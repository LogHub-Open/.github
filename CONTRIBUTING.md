# Contribuindo com a LogHub Open

Obrigado pelo interesse em contribuir! Siga as orientações abaixo para garantir uma boa experiência para todos.

## Como contribuir

1. **Fork** o repositório desejado
2. Crie uma branch descritiva: `git checkout -b feat/minha-feature`
3. Faça seus commits seguindo o padrão [Conventional Commits](https://www.conventionalcommits.org/pt-br/)
4. Abra um **Pull Request** com uma descrição clara das mudanças

## Padrão de commits

```
feat: adiciona nova funcionalidade
fix: corrige bug em X
docs: atualiza documentação
chore: ajustes de configuração
```

## Reportando bugs

Abra uma issue com:
- Descrição clara do problema
- Passos para reproduzir
- Comportamento esperado vs atual
- Versão do sistema/ambiente

## Sugerindo melhorias

Abra uma issue com a tag `enhancement` descrevendo:
- O problema que a melhoria resolve
- A solução proposta
- Alternativas consideradas

## Código de Conduta

Este projeto adota nosso [Código de Conduta](CODE_OF_CONDUCT.md). Ao participar, você concorda em respeitá-lo.
