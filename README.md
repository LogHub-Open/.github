# .github — LogHub Open

Repositório central da organização [LogHub-Open](https://github.com/LogHub-Open).

## Estrutura

| Arquivo/Pasta | Finalidade |
|---|---|
| `profile/README.md` | Página pública da organização no GitHub |
| `CONTRIBUTING.md` | Guia de contribuição para todos os projetos |
| `CODE_OF_CONDUCT.md` | Código de conduta da comunidade |
| `SECURITY.md` | Política de segurança e reporte de vulnerabilidades |
| `ISSUE_TEMPLATE/` | Templates padrão de issues |
| `PULL_REQUEST_TEMPLATE.md` | Template padrão de Pull Requests |

Os arquivos neste repositório são aplicados automaticamente a todos os repositórios da organização que não possuem seus próprios arquivos equivalentes.
